# Libft

A C library created from scratch, containing recoded standard library functions and more.

## Installation:

`git clone https://github.com/fpetras/libft.git && cd libft && make`

## Usage:

`gcc main.c ... libft.a`\
or\
`mv libft.a /usr/local/lib && gcc main.c ... -lft`
